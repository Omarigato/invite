# Shaqyrtu.kz — Архитектура сайта

## Что это

Shaqyrtu — казахстанский сервис создания цифровых приглашений на мероприятия (тои). Пользователь выбирает дизайн, заполняет тексты, загружает фото и получает ссылку-приглашение для отправки гостям через WhatsApp/Telegram/SMS.

**Домен:** `https://shaqyrtu.kz`
**Разработчик:** [abmco.kz](https://abmco.kz)
**Языки:** KZ (казахский), RU (русский)

---

## Технологический стек

| Слой | Технология |
|------|-----------|
| Framework | **Nuxt 3** (Vue 3, SSR/SSG) |
| UI-библиотека | **Nuxt UI v3** |
| CSS | **Tailwind CSS** (utility-first) |
| Цветовая система | **OKLCH** (`oklch(67.3% 0.182 276.935)` — primary indigo) |
| Шрифты | SF Pro Text (локальный .ttf), Comfortaa, Cormorant Garamond, Noto Serif, Jost, Great Vibes |
| 3D/Анимации | **three.js** + WebGL (Floral-дизайн) |
| Аналитика | **PostHog** (autocapture, web-vitals, dead-clicks, surveys) |
| SEO | JSON-LD (Schema.org), OpenGraph, Twitter Cards |
| PWA | Service Worker, `site.webmanifest` |
| Оплата | **Kaspi.kz** (QR), **FreedomPay** (карты) |
| Мессенджер | **Telegram Bot** (`@shaqyrtu_kz_abmco_bot`) |

## Модалка "Шақыруды белсендіру"

Открывается при нажатии "Бөлісу".

### Этап 1: Кастомная ссылка
- "Жеке сілтеме" (Личная ссылка)
- Поле: `shaqyrtu.kz/i/{slug}` (placeholder: "мысалы: aida-daniyar")
- Срок: 2 месяца, изменить 1 раз
- Кнопка **"Тексеру"** (Проверить)
- Ссылка "Керегі жоқ, өткізіп жіберу" (Пропустить)

### Этап 2: Промокод
- "Промокодыңыз бар ма?"
- **"Иә, промокодым бар"** → ввод кода
- **"Жоқ, төлеуге өту"** → к оплате

### Этап 3: Оплата

**Цена:**
- Без промокода: **5 000 ₸**
- Со спонсором: ~~5 000 ₸~~ **1 000 ₸** (-80%)

**Способы:**

| Способ | Кнопка | Ссылка |
|--------|--------|--------|
| Kaspi.kz | 🔴 "Kaspi.kz арқылы төлеу" | `pay.kaspi.kz/pay/{id}` |
| Картой | ⬜ "Картамен төлеу" | `customer.freedompay.kz` |
| Спонсор | 🟡 "−80% Серіктес демеуімен" | — |

**Дополнительно:**
- "Сілтемені көшіру" — копировать ссылку
- "Бөлісу" — поделиться
