# AGENTS.md

## What this repo is

A saved (offline) copy of the shaqyrtu.kz landing page — a Nuxt UI-based site for an electronic invitations service in Kazakhstan. This is **not a source code repository**. The HTML was exported from a browser (`saved from url=(0014)about:internet`).

## Key facts

- **Single HTML file**: `Shaqyrtu — сервис электронных приглашений в Казахстане.html` — the full page with inlined CSS/JS.
- **Static assets folder**: `Shaqyrtu — сервис электронных приглашений в Казахстане_files/` — images (`.jpg`, `.png`), CSS modules (hashed names like `Altyn.CCYmLdXi.css`), and PostHog analytics JS snippets.
- **Framework**: Nuxt 3 with Nuxt UI (indigo/blue theme, oklch color system).
- **Language**: Russian (`lang="ru"`), Kazakh branding ("Shaqyrtu").
- **Analytics**: PostHog (autocapture, surveys, web-vitals, dead-clicks, tracing-headers).

## No build/test/lint

There is no `package.json`, no source code, no build pipeline, no tests, no CI. This repo is a static reference snapshot — not an active development project.
